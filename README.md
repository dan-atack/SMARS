## To start the project up in local development mode:

Frontend:

    cd frontend/

    npm run start

Backend:

    cd backend/

    npm run dev

Database:

    * Ensure mongo service is running

Once the stack has started, you should be able to play the game by going to http://localhost:1234 in your internet browser
